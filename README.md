# Raspberry Pi IoT Projects Collection 

This repository contains multiple beginner-to-intermediate level IoT projects developed using Raspberry Pi, sensors, LEDs, and GPIO programming in Python. These projects demonstrate hardware interfacing, automation logic, and real-time sensor control.

The projects were developed as part of my IoT learning journey and practical implementation experience.

📁 Projects Included
LED Control (led.py)

Basic LED ON/OFF control using Raspberry Pi GPIO.

Demonstrates digital output control

Suitable for beginners to understand GPIO programming

Concepts: GPIO output, digital control

Button Control (Button.py)

Controls an LED using a push button.

Detects button press and triggers an output action

Concepts: GPIO input, event detection, user input control

Motion Sensor System (motion sensor.py)

Uses a PIR motion sensor to detect movement.

Turns ON LED when motion is detected

Applicable in security and automation systems

Concepts: PIR sensor, motion detection, automation

Distance Sensor with LED Indicator (Distance sensor.py)

Uses an ultrasonic sensor to measure distance.

LED indicates object proximity

Can be applied in obstacle detection systems

Concepts: Ultrasonic sensor, distance measurement, real-time response

Traffic Signal System (Traffic signal.py)

Simulates a traffic signal system using LEDs.

Implements Red, Yellow, and Green signal sequence

Concepts: Timing control, sequential logic, real-world simulation

Hardware Components Used

Raspberry Pi

LEDs

Push Button

PIR Motion Sensor

Ultrasonic Distance Sensor (HC-SR04)

Resistors

Jumper wires

Software Requirements

Python 3

Raspberry Pi OS

GPIO libraries: RPi.GPIO or gpiozero
